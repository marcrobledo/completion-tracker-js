Completion Tracker JS

Completion Tracker JS is a JavaScript library designed to easily create interactive web lists.
It offers a simple and easily integrable solution, allowing anyone to quickly build their own progress tracking web app.

Initially, I developed it solely for personal use because I enjoy achieving 100% completion in my favorite games that I replay, and I needed a practical solution to ensure I wasn't missing any collectibles (until then, I had been using pen and paper!).
After finishing it, I decided to release it to the public in case it might be useful to someone else.

You'll find more information about the library and its usage instructions on its GitHub repository.
